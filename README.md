# Insta-Api-Clone
- Attempt to clone some of functionalities of Instagram
- Frontend in React.js

## Tools and Tech.
- Python
- Django
- Django Rest Framework
- PostgreSQL Database
- Web Sockets
- Django Channels
- Redis Server
- pip
- virtual env
- PyCharm

## Features
- User Management
- Post Management
- Follow/Unfollow
- Like, Comment
- Chat
- Story for 24 hours
- Archived Stories

## Development
- You should have PostgreSQL and Redis-Server running on your machine
- Create Virtual Environment
- Install Following Dependencies
  - django, pillow, channels, channels_redis, rest-framework
- Configure Database, Username and Password in .env File
- Run following Commands
```
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
