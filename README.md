# Insta-Api-Clone
- Attempt to clone some of functionalities of Instagram
- Frontend in React.js

## Tools and Tech.
- Python
- Django
- Django Rest Framework
- PostgreSQL Database
- Web Sockets
- Django Channels
- Redis Server
- pip
- virtual env
- PyCharm

## Features
- User Management
- Post Management
- Follow/Unfollow
- Like, Comment
- Chat
- Story for 24 hours
- Archived Stories
